# 🚗 Vehicle Detection using YOLOv11

## 📌 Project Overview
This project implements a **Vehicle Detection System** using the **YOLOv11 (You Only Look Once)** object detection model. The model is trained on a custom dataset to detect multiple vehicle classes such as cars, buses, trucks, motorcycles, etc.

The system performs:
- Dataset preprocessing  
- Model training  
- Performance evaluation  
- Visualization of results  
- Testing on new images  

---

## 🎯 Objectives
- Build an efficient vehicle detection model using YOLOv11  
- Achieve high accuracy in real-time object detection  
- Evaluate performance using standard metrics like Precision, Recall, mAP, and F1-score  
- Visualize training and evaluation results  

---
